## Events
### event-name1
이 이벤트에 대한 한 줄 내외의 간단한 설명을 작성한다.
- **매개변수** data(Type) 매개변수에 대한 간단한 설명
- data의 데이터
  |Field|Type|Description|
  |---|---|---|
  |...|Boolean|...|
- data 데이터 예시
  ```json
  {
    "field1": true,
    "field2": "hello,world"
  }
  ```
 _사용 예시:_
 ```html
 <template>
  <div></div>
 </template>
 
 <script>
 </script>
 ```
